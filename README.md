# AI Exam Proctoring System

## Overview

AI Exam Proctoring System is an AI-powered online examination monitoring system developed to ensure secure and fair online exams. The project uses Artificial Intelligence and Machine Learning techniques to monitor student activities during online examinations.

The system includes features such as:

* Face Detection and Recognition
* Eye Gaze Tracking
* Head Pose Detection
* Voice Monitoring
* Object Detection
* Keystroke Monitoring
* Federated Learning
* Plagiarism Detection
* Real-time Exam Monitoring

## Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/Rayeesa-Anjhum/AI_Exam_Proctoring_System.git
```

### Step 2: Navigate to the Project Folder

```bash
cd AI_Exam_Proctoring_System
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Start the Server

```bash
python app.py
```
